# Users App

Тестовое задание — отображение списка пользователей из БД через fullstack-связку.

## Стек
- **Frontend:** Angular
- **Backend:** ASP.NET Web API
- **БД:**  PostgreSQL
- **ORM:** Entity Framework 

## Запуск

### Бэкенд
1. Перейти в папку `api`
2. Настроить строку подключения к БД в `appsettings.json`
3. Запустить: `dotnet run`

### Фронтенд
1. Перейти в папку `client`
2. Установить зависимости: `npm install`
3. Запустить: `ng serve`
4. Открыть в браузере: `http://localhost:4200`
